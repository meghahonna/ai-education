# AI Education Workshop

A practical introduction to AI for non-technical staff. Covers core AI concepts, hands-on applications for everyday operations and data analysis, role-specific use cases, and responsible AI guidelines. Designed to be accessible and adaptable for any organization.

## What's Inside

| File | Description |
|------|-------------|
| `AI_GenAI_Golf_Management.pptx` | Workshop slide deck — AI fundamentals, role-specific applications, and responsible use guidelines |
| `AI_GenAI_Golf_Management.key` | Keynote version of the slide deck |
| `context-engineering-playground.html` | Interactive single-page tool for building structured prompts and visualizing context rot — open in any browser, no setup needed |
| `prompt-template.md` | 10-section prompt engineering template with fill-in scaffolding and worked examples (grant writing, donor data analysis, executive reporting) |
| `synth_data/` | Synthetic donor dataset for hands-on exercises — no real PII |
| `ollama-chat/` | Sample application for running a local LLM chat using Ollama |

## Quick Start

1. **Workshop deck** — Open the `.pptx` or `.key` file and present as-is, or customize for your org
2. **Prompt template** — Share `prompt-template.md` as a handout or reference doc
3. **Context playground** — Open `context-engineering-playground.html` in a browser — load a sample prompt, paste text into the conversation history 10x, and watch the context rot meter climb
4. **Hands-on exercise** — Use the synthetic donor data in `synth_data/` with any AI tool to practice data analysis prompting
